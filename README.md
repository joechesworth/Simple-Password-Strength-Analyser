# Simple-Password-Strength-Analyser
A simple password strength analyser using python. It contains a simple GUI that you can enter your password into and it will score it based on a number of factors, including length, case, characters. It rates it using an entropy algorithm and also checks if it contains any real words or common phrases.
It uses the NLTK library to import a large word list which contains a large amount of common english words.
Finally it will provide you with feedback and give you recommendations on how you can improve it.
